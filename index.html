Introduction

Python is what is known as an interpreted programming language. Meaning, the computer compiles the source code ‘on the fly’, unlike some other languages, such as C and Java, that make compiling the program a separate step in the development process. Python is a very beginner-friendly language to learn and scales up quickly to handle more complex computations.

This documentation will assume the reader has basic computer skills and is familiar with searching the internet, downloading files and installing apps.

Install and Set Up

In order to run Python scripts on your computer, you must first install Python. The download for the latest version of Python can be found here.

Python install by OS:

  • Windows / Mac OS: Follow the above link to install the latest version for your OS. For Windows, be sure to check the box next to Add to path.

  • Linux: Most Linux distrubtutions come with some version of Python already installed. To check what Python version is installed, type python in the terminal window. The current latest version is Python 3, to install it type sudo apt install python3 in the terminal window.
